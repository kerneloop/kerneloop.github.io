Comandos Linux

Lista de comandos útiles

En esta seccion se indicaran diferentes comandos de linux con los que puedes trabajar tanto en tu día a día como para tareas específicas, disfruta del contenido y estate siempre actualizado!!

Formato

(Paquetería) - (comando) -> (función)

Para instalar cualquier paquete utilizaremos el comando:

sudo apt install {paquete} -y

  • default - ls -> Lista documentos el el directorio señalado, con la flag “-a” veremos tambien los ocultos y algunos detalles más.

  • net-tools - arp -> Manejo de tablas ARP del equipo, utilizando la flag “-s” añadimos registros, con “-d” los eliminamos y con “-a” los listamos.

  • Imagemagick - convert -> Convierte un formato de imagen a otro, tambien puedes redimensionar imágenes.
